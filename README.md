# Base Template - Website Directory Structure

This is a template directory structure for websites in the ZZZWebsites collection.

## Directory Structure

```
base-template/
├── .claude/              # Claude AI configuration and session files
├── css/                  # Stylesheets and CSS files
├── docs/                 # Documentation, plans, and guides
├── images/               # Image assets and media files
├── pages/                # Content pages and HTML files
├── scripts/              # JavaScript, Python, and utility scripts
└── tools/                # Tools and utilities
```

## Directory Purposes

### `.claude/`
Claude-specific configuration files, memory, and session notes.

### `css/`
All stylesheet files (.css). Organize by:
- `main.css` - Main stylesheet
- `responsive.css` - Media queries and responsive design
- `theme.css` - Theme/color variables
- Subdirectories for component-specific styles if needed

### `docs/`
Documentation and planning files:
- Plans and specifications
- Architecture notes
- Implementation guides
- Research and reference materials

### `images/`
All image assets:
- `img/` - Regular images (PNG, JPG, SVG, etc.)
- `icons/` - Icon files
- `pdfs/` - PDF assets if applicable
- `screenshots/` - Screenshot references

### `pages/`
HTML content pages and page templates:
- HTML files for each page
- Index pages
- Template snippets
- Archived or old pages in `oldpages/` subdirectory

### `scripts/`
Executable scripts and utilities:
- JavaScript files
- Python scripts
- Build/automation scripts
- Utility functions

### `tools/`
Standalone tools and utilities:
- Helper applications
- Build tools
- Configuration utilities
- Scripts for site management

## Usage

Use this structure as a template when creating new websites in the ZZZWebsites directory. Adapt as needed for specific project requirements.
